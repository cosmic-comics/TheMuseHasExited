# The Muse has Exited

## Development

1. Get the source code:

```sh
git clone https://github.com/cosmic-comics/TheMuseHasExited && cd TheMuseHasExited
```

2. Install [Ren'py](https://renpy.org)

3. Start the process:

```sh
renpy .
```
